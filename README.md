## Installing / Getting started

Python3 must be already installed

```shell
git clone https://github.com/SheepCoders/excel-to-html-tb.git
cd excel-to-html-tb
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
```
